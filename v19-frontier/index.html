Chicago Field Guide — Iteration 19

Western / Frontier

Wanted Poster · Desert Sand · Rust Red · Rye + Arvo · 19th-Century Land Survey

Concept

Chicago as a frontier city — which it was. Founded as Fort Dearborn in 1803, it was a trading post at the edge of the known world. The railroads made it the nexus of the American West. The stockyards defined it for a century. This is the city Carl Sandburg called "Hog Butcher for the World."

The constraint: wanted poster typography using Rye for display and Arvo (a slab serif) for body. Both are serifs. Both feel like wood type. The combination reads like a 19th-century broadside — authoritative, rough-edged, hand-set.

Design Decisions

Double-frame spot cards Cards use border-left for category color + inset 0 0 0 3px rgba(184,144,104,0.1) inner shadow to create a subtle double-frame effect — like a printed label with a decorative rule border.
Hard displacement shadows everywhere 3px 3px 0 rgba(44,26,8,0.2) on cards, buttons, popups. The same letterpress-print shadow language as V11 (UChicago), but warmer and more aged — saddle brown instead of maroon.
Slab-serif body text Arvo is a slab serif — the same structural DNA as the type on 19th-century wanted posters and railroad schedules. This is the only version where even the paragraph text feels like a historical document.
19th-century survey map sepia(0.6) saturate(0.55) brightness(0.92) contrast(1.02) — heavier sepia than any other version. The map reads like a GLO survey section plat from 1840, before the city was fully platted.
