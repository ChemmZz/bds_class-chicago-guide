01 — Concept

BRUTALISM AS METHODOLOGY

Brutalist architecture exposes structure honestly — concrete is concrete, steel is steel, nothing is hidden behind decorative facing. This iteration applies the same philosophy to the web interface: the layout grid is visible, borders are thick and declarative, no element is softened by radius or gradient.

"ORNAMENT IS CRIME. THE GRID IS THE DESIGN. THE BORDER IS THE WALL."

The reference is Swiss brutalism and the raw-web brutalist aesthetic — sites that deliberately resist the visual conventions of "polished" UI. The result should feel intentional and disciplined, not unfinished. Brutalism is not minimalism. It is maximum honesty about structure.

02 — Palette

WHITE, CONCRETE, BLACK — NOTHING ELSE

Three values: pure white (#ffffff), raw concrete (#c8c8c8), pure black (#000000). No hues. No saturation. The only color decision is which of the three applies to each surface — and the answer is always structural: white for content areas, concrete for backgrounds and inactive states, black for all borders and structural lines.

#ffffff White
#c8c8c8 Concrete
#4a4a4a Shadow
#000000 Black

Tag colors in the map markers are all black — every marker is identical except its number. The route polyline is black. The focused card inverts: black background, white text. The system has exactly one bit of state per element.

03 — Typography

BEBAS NEUE, SPACE GROTESK, SPACE MONO

Bebas Neue is all-caps, condensed, structural — the brutalist display face. Space Grotesk carries body text with geometric neutrality. Space Mono aligns with the technical, utilitarian register of labels and addresses. All three are geometric, all three resist ornamentation.

No italic, no script, no serif. Letter-spacing is used structurally — not decoratively — to adjust optical weight in labels and headings.

04 — Map

RAW OSM TILES — NO FILTER

This is the only iteration with no CSS filter on the map tiles. Raw OpenStreetMap tiles are used as-is — their own visual language is treated as found material, the way a brutalist architect leaves concrete formwork marks visible in the finished surface.

/* No filter applied */
.leaflet-tile-pane {
  /* raw OSM as found — the map's own material logic */
}
