01 // Concept

The city as terminal output

Cyberpunk treats the guide as a data terminal — everything is output, addresses are coordinates, descriptions are readouts. The visual reference is the CRT monitor: sharp edges, phosphor glow, scan lines, no organic curves anywhere.

// CITY DATA LOADED. 25 NODES. ROUTE COMPUTED. AWAITING INPUT.

Orbitron provides geometric display type that reads as future-tech without being illegible. Share Tech Mono handles everything else — all body text, all labels, all addresses — producing an interface that looks more like a command-line tool than a web page.

02 // Palette

Black, acid green, magenta — nothing else

Three colors. True black background. Acid green (#00ff41) is the primary data color — all readable information, all interactive states, all route lines. Magenta (#ff00ff) is used exclusively as alert or accent — tip labels, hover states on secondary elements.

#050505 BG
#00ff41 Green
#ff00ff Magenta
#1a0033 Deep Purple
#111111 Surface

The CSS scanline effect is a repeating linear gradient with a 2px period — alternating between transparent and a very faint rgba(0,0,0,0.15) — creating the CRT texture without any image assets.

03 // Map

Maximum darkness — just the street grid

The map filter is pushed to maximum here: brightness so low that only the street grid survives as faint phosphor traces on black. The effect is intentional — the markers become the only thing that matters, the map just provides spatial context.

.leaflet-tile-pane {
  filter: brightness(0.2) saturate(0) invert(1) hue-rotate(180deg);
  // near total black, street grid as faint traces
}
