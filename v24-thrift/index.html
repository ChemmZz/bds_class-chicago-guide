Concept

V24 is a guide to Chicago's vintage and thrift economy — the record shops, clothing resalers, antique markets, and curiosity stores that define the city's alternative retail landscape. The itinerary is anchored in Wicker Park's Milwaukee Avenue corridor (Reckless Records, Knee Deep Vintage, Ragstock, Dusty Groove) and fans out to Logan Square, Andersonville, and Rogers Park.

The palette stays Pilsen — V18's four-accent marigold/magenta/teal/cobalt system, which reads equally well against cream parchment and dark map tiles. The thrift economy and the Pilsen color system share a sensibility: vibrant, maximal, nothing wasted.

Design Decisions

Price tag SVG header Two translucent vintage price tags flank the title — a marigold main tag and a smaller magenta accent tag, connected by a dashed twine string. Each rotated slightly in opposite directions for a casual, thrown-on-the-rack feel. Same CSS absolute-positioning pattern as V21–V23, opacity 0.42 so the Pilsen rainbow border-image remains visible.
Thrift-native tag taxonomy V24 replaces the generic tag set (Architecture, Jazz, Doors) with thrift-specific categories: Vintage, Records, Antiques, Clothing, Nightlife, Café. Each maps to a Pilsen accent color — teal for records, cobalt for antiques, magenta for clothing, marigold for vintage.
Pilsen palette as system Starting with V24, all future iterations use V18's palette as the base template. The four-accent Pilsen system is flexible enough to carry different content themes while maintaining visual coherence across the gallery.
Geographic spread 25 spots across Wicker Park, Logan Square, Lincoln Park, Lakeview, Andersonville, Edgewater, Rogers Park, River North, and the West Loop — a proper cross-city thrift itinerary with the nearest-neighbor route optimizer finding an efficient path.
