Dark Chicago

Chicago municipal flag palette — deep navy, cyan, red. Moody and editorial. Dark OSM tiles via CSS invert. Playfair Display headings, Space Mono labels. Glows, pill buttons, maximum contrast.

Japandy

Japanese minimalism (ma — the meaningful interval) meets Scandinavian warmth. Warm off-white ground, ink-blue accent. Cormorant Garamond at 300 weight. Light map tiles, outline markers, rectangular buttons. Maximum restraint.

Gotham Noir

Near-black ground, bat signal yellow accent. Bebas Neue at display sizes, Share Tech Mono for surveillance-terminal labels. CSS map filter pushed to brightness(0.28) — barely a city, barely a map. Gritty, noir, unforgiving.
