01 — Concept

THE BRIEF: THE CITY AT NIGHT

V1 was editorial and civic — navy, cyan, the Chicago flag. V2 stripped everything back to warm neutrals and whisper-weight type. V3 asks a different question: what if the guide felt like surveillance? Like a dossier? Like something slid under a door in the rain?

Batman's Gotham is fictional, but Chicago earns its noir credentials honestly — Al Capone's old booth at the Green Mill, the Gothic Revival Tribune Tower, underground dance clubs below rock venues. The city has always had a shadow side. This iteration makes that explicit.

SAME DATA. DIFFERENT MASK.

The constraint: no new spots, no new tags, no new routing logic. The data is identical. The design language does all the work. Typography, color, filter intensity — that's it.

02 — Design

NEAR-BLACK + BAT SIGNAL YELLOW

The palette reduces to two poles: near-black (#080808) and bat signal yellow (#f5c518). Everything else — tag colors, text, borders — exists between those two extremes. The yellow appears only where it matters: the header border, active filter glows, route badges, the polyline cutting across the dark map.

Tag colors are deliberately muted — weathered stone for architecture, cold blue for music, deep purple for nightlife. They're desaturated enough not to fight the dark ground but vivid enough to distinguish categories at a glance. Active filter buttons don't fill with color; they glow with it — a thin border in tag color with a color-matched halo shadow.

Bebas Neue at display sizes reads like a headline torn from a newspaper. Combined with Share Tech Mono for addresses and labels — a surveillance terminal aesthetic — the guide feels like a classified brief rather than a lifestyle app. Barlow fills the middle ground with just enough warmth to keep it readable.

03 — Logic

THE SAME ALGORITHM. THE SAME DATA.

Filtering is additive: selecting multiple tags returns spots matching any of them. The route algorithm is a nearest-neighbor heuristic starting from Millennium Park. For each step it selects the unvisited spot closest to the current position by haversine distance — the spherical-earth calculation that accounts for longitude distortion at Chicago's latitude (~42°N).

1
Millennium Park (anchor)
2
Nearest unvisited spot
3
Next nearest from #2…
N
All filtered spots covered

The route isn't mathematically optimal — that's NP-hard — but it's geographically sensible. Nearest-neighbor gets within 15–20% of optimal for city-scale datasets with no perceptible lag. The yellow dashed polyline updates live on every filter change.

04 — The Map

SURVEILLANCE GRADE — THE DARKEST FILTER YET

The same CSS invert trick from V1, but pushed further. V1 used brightness(0.55). V3 drops to brightness(0.28) and desaturates more aggressively — the map reads like a satellite thermal image, barely showing street structure beneath the dark ground.

.leaflet-tile-pane {
  filter: brightness(0.28) saturate(0.15) invert(1) hue-rotate(180deg); /* surveillance mode */
}

Route markers are the same technique as V1 — Leaflet L.divIcon with tag-colored circles and a yellow border ring. The route polyline switches from cyan to bat signal yellow. Popups gain a yellow top border — the only element that breaks the dark container pattern.
