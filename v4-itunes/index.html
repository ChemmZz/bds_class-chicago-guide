01 — Concept

The brief: what if Apple designed a city guide in 2003?

Mac OS X 10.2 Jaguar shipped in August 2002. iTunes 4 launched in April 2003. The Aqua interface — brushed metal titlebars, candy-colored aqua buttons, pinstripe backgrounds, thick scrollbar thumbs — was the dominant design language of the era. Everything felt physical, tactile, glossy.

V1 was civic and editorial. V2 was Japandy restraint. V3 was Gotham noir. V4 asks: what if the Chicago Field Guide had shipped as a Mac OS X application in 2003? Same spots, same algorithm — but rendered in the language of Source Lists, Aqua buttons, and brushed metal.

"It looks like it belongs in the dock."

The interface borrows directly from iTunes: a source-list sidebar with colored dot icons, track-list rows that highlight blue on selection, a "Now Playing" pill at the bottom of the map. The header is a brushed-metal gradient with a repeating-horizontal-stripe texture. Everything has a shadow and a highlight.

02 — Design

Brushed metal, Aqua blue, candy dots

The palette is drawn from the Aqua HIG (Human Interface Guidelines) of the era. The background is a warm silver (#e2e2e2) with a subtle pinstripe. The primary accent is iTunes blue (#3478f6) — the same blue that appeared on every progress bar, selection highlight, and button from 2001 to 2007.

Tag filter buttons become iTunes-style source list rows — full-width, with a small colored dot at left (like playlist icons). Selected rows get the characteristic blue gradient: linear-gradient(#5a9cf8, #3478f6). Spot cards become track list rows: compact, alternating zebra stripe, blue highlight on focus with white text throughout.

Cabin is the closest Google Fonts equivalent to Myriad Pro — the typeface Apple used in nearly every product from 2002 to 2017. Combined with Inconsolata for addresses and code-like labels, the typographic register reads as "Apple application" rather than "website."

03 — Logic

Same algorithm, different UI metaphor

The routing and filtering logic is identical across all four versions. Nearest-neighbor from Millennium Park, additive OR filtering, Leaflet.js with OSM tiles. The only map change in V4: a light saturate(0.75) brightness(0.97) filter — just enough to give the OSM tiles the slightly warm, slightly dull look of an early 2000s web map. No inversion, no dark mode.

The route polyline is iTunes blue (#3478f6) — dashed, 2.5px, 70% opacity. The "Now Playing" info bar at the bottom of the map is a silver pill with a gradient border and an inset shadow — the Aqua control metaphor applied to a route counter.

04 — Technique

Faking brushed metal with pure CSS

The brushed metal effect is a CSS layered background — a repeating-linear-gradient of transparent and near-white stripes (mimicking the horizontal brush marks) layered over a vertical silver gradient. No images, no SVG.

.guide-header {
  background:
    /* brush stripes */
    repeating-linear-gradient(
      90deg,
      transparent 0 3px,
      rgba(255,255,255,0.04) 3px 6px
    ),
    /* silver gradient */
    linear-gradient(180deg,
      #d8d8d8 0%, #b8b8b8 55%, #cccccc 100%
    );
}

Aqua buttons use three stacked effects: a multi-stop gradient (light → mid → dark top-to-bottom), an inset 0 1px 0 white highlight, and a drop shadow. The thick Aqua scrollbar thumb is styled with ::-webkit-scrollbar-thumb — on hover it turns the signature iTunes blue gradient.
